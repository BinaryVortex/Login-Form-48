# Login Form

A clean, responsive login form built with plain HTML, CSS and JavaScript.

> Login Form Built Using HTML, CSS And Javascript.


## Preview

![Login Form Screenshot](https://raw.githubusercontent.com/BinaryVortex/Login-Form-48/main/Screenshot%202024-07-25%20115609.png)


## Features

- Simple, modern-looking login UI.
- Built with semantic HTML and CSS (no frameworks).
- Lightweight and easy to customize.


## Files

- `index.html` — the login form markup.
- `style.css` — styles for layout, colors and responsiveness.
- `Screenshot 2024-07-25 115609.png` — project preview image.
- `digital-art-with-city-landscape-architecture_23-2151065718.jpg` — background/art asset included in the repo.


## How to use

1. Clone or download the repository.
2. Open `index.html` in your browser (double-click or use a local web server).

Optional: Serve with a simple HTTP server from the project folder:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```


## Customization

- Edit `style.css` to change colors, spacing, or fonts.
- Update `index.html` to add fields or integrate with your backend.


## Notes

This is a front-end template only. To make the login functional you should connect the form to a secure backend that handles authentication, password storage, and session management.
